# Better Auth 1.2 – Stripe、验证码、API 密钥、团队、Init CLI 等功能

升级请运行：

```package-install
npm install better-auth@1.2
```

---

### **Stripe 插件（测试版）**

集成 Stripe 用于客户管理、订阅和 Webhook。

```package-install
npm install @better-auth/stripe
```

```ts title="auth.ts"
import { stripe } from "@better-auth/stripe"; // [!code highlight]

export const auth = betterAuth({
  plugins: [
    stripe({
      // [!code highlight]
      createCustomerOnSignup: true, // [!code highlight]
      subscription: { // [!code highlight]
        enabled: true, // [!code highlight]
        plans: [// [!code highlight]
          { // [!code highlight]
            name: "pro", // [!code highlight]
            priceId: "price_1234567890", // [!code highlight]
          }, // [!code highlight]
        ], // [!code highlight]
      }, // [!code highlight]
    }), // [!code highlight]
  ],
});
```

更多信息请参阅 [Stripe 插件文档](/docs/plugins/stripe)。

### **验证码插件**

通过 Google reCAPTCHA 和 Cloudflare Turnstile 保护你的认证流程。支持注册、登录和密码重置。

```ts title="auth.ts"
import { captcha } from "better-auth/plugins";

const auth = betterAuth({
  plugins: [
    // [!code highlight]
    captcha({
      // [!code highlight]
      provider: "cloudflare-turnstile", // 或 "google-recaptcha" // [!code highlight]
      secretKey: process.env.TURNSTILE_SECRET_KEY!, // [!code highlight]
    }), // [!code highlight]
  ], // [!code highlight]
});
```

更多信息请参阅 [验证码插件文档](/docs/plugins/captcha)。

### **API 密钥插件**

生成和管理带有速率限制、过期和元数据的 API 密钥。支持通过 API 密钥创建会话。

```ts title="auth.ts"
import { apiKey } from "better-auth/plugins";

const auth = betterAuth({
  plugins: [apiKey()],
});
```

更多信息请参阅 [API 密钥插件文档](/docs/plugins/api-key)。

### **团队/子组织**

组织现在可以拥有团队或子组织。

```ts title="auth.ts"
const auth = betterAuth({
  plugins: [
    organization({
      teams: {
        enabled: true,
      },
    }),
  ],
});
```

更多信息请参阅 [组织插件文档](/docs/plugins/organization#teams)。

### **Init CLI**

CLI 现包含 `init` 命令，可将 Better Auth 添加到你的项目中。

```bash title="terminal"
npx auth init
```

### **用户名**

- 新增 `displayName` 用于不区分大小写的查找，同时保留原格式。
- 内置校验。

<Callout type="info">
  如果你正在使用用户名插件，请确保在你的 schema 中添加 `displayName` 字段。
</Callout>

### **组织**

- **用户可拥有多个角色** – 允许为用户分配多个角色。

### **管理员插件**

- 在管理员插件中管理角色和权限。[了解更多](/docs/plugins/admin)
- `adminUserIds` 选项，授予特定用户管理员权限。[了解更多](/docs/plugins/admin#usage)

## ✨ 核心增强功能

- 过期验证数据的**自动清理**
- 更完善的 Google One Tap 集成，支持 JWT 验证和改进的提示处理
- 基于电话的密码重置功能
- 提供者控制选项：
  - 禁用特定提供者的注册
  - 禁用特定提供者的隐式注册
  - 控制默认权限范围，并允许请求时指定自定义权限范围
- 增强数据库钩子，包含更多上下文信息

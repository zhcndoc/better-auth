---
title: SolidStart 集成
description: 将 Better Auth 集成到 SolidStart 中。
---

在开始之前，请确保你已经配置好了 Better Auth 实例。如果还没有，请查看 [安装指南](/docs/installation)。

### 挂载处理器

我们需要将处理器挂载到 SolidStart 服务器。在 `/routes/api/auth` 文件夹内的 `*auth.ts` 文件中添加以下代码。

```ts title="*auth.ts"
import { auth } from "~/lib/auth";
import { toSolidStartHandler } from "better-auth/solid-start";

export const { GET, POST } = toSolidStartHandler(auth);
```

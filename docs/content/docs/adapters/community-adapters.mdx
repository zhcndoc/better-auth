Better Auth 通过适配器连接到数据库。除了我们维护的官方适配器之外，社区还构建了许多其他适配器，以支持更多数据库。我们鼓励您创建自定义适配器，并且也许还能被添加到列表中！

## 创建您自己的适配器

要创建您自己的自定义适配器，请先阅读我们的[创建数据库适配器](/docs/guides/create-a-db-adapter)指南。如果您想与社区分享您的适配器，请提交拉取请求以将其添加到此列表中。

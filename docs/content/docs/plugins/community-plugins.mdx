插件使 Better Auth 具备灵活性和可扩展性。除了我们维护的官方插件外，社区还针对各种使用场景构建了许多插件。我们鼓励您创建自定义插件，并可能将其加入此列表！

## 创建您自己的插件

要创建自己的自定义插件，请先阅读我们的[创建您的第一个插件](/docs/guides/your-first-plugin)指南。如果您想与社区分享您的插件，请提交拉取请求以将其添加到此列表中。

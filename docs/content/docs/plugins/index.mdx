Better Auth 提供了 50 多个插件，扩展框架，增加额外的身份验证功能、支付集成、安全工具等。

## 身份验证

| Plugin                                               | Description                                                |
| ---------------------------------------------------- | ---------------------------------------------------------- |
| [双重认证](/docs/plugins/2fa)       | 通过双重认证增强应用安全性 |
| [通行密钥](/docs/plugins/passkey)                     | WebAuthn/通行密钥认证                            |
| [魔法链接](/docs/plugins/magic-link)               | 无密码邮件魔法链接认证               |
| [邮箱一次性密码](/docs/plugins/email-otp)                 | 基于邮箱的一次性密码认证               |
| [手机号码](/docs/plugins/phone-number)           | 手机号码认证                                |
| [匿名用户](/docs/plugins/anonymous)                 | 匿名/访客会话支持                            |
| [用户名认证](/docs/plugins/username)                   | 基于用户名的认证                              |
| [一键登录](/docs/plugins/one-tap)                     | Google 一键登录登录                                     |
| [以太坊登录](/docs/plugins/siwe)          | 以太坊钱包认证                             |
| [通用 OAuth](/docs/plugins/generic-oauth)         | 使用任意 OAuth 提供程序进行认证                 |
| [多会话](/docs/plugins/multi-session)         | 每个用户多个并发会话                      |
| [最后登录方式](/docs/plugins/last-login-method) | 追踪最后使用的认证方式                  |

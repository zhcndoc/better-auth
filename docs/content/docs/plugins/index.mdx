Better Auth 提供了 50 多个插件，扩展框架，增加额外的身份验证功能、支付集成、安全工具等。

| 插件 | 描述 |
| --- | --- |
| [双重认证](/docs/plugins/2fa) | 通过双重认证增强您的应用安全性 |

| [魔术链接](/docs/plugins/magic-link) | 无密码邮箱魔术链接认证 |
| [邮箱一次性密码](/docs/plugins/email-otp) | 基于邮箱的一次性密码认证 |
| [手机号](/docs/plugins/phone-number) | 手机号码认证 |
| [匿名](/docs/plugins/anonymous) | 匿名/访客会话支持 |
| [用户名](/docs/plugins/username) | 基于用户名的认证 |
| [一键登录](/docs/plugins/one-tap) | Google 一键登录 |
| [以太坊登录](/docs/plugins/siwe) | 以太坊钱包认证 |
| [通用 OAuth](/docs/plugins/generic-oauth) | 使用任意 OAuth 提供者认证用户 |
| [多会话](/docs/plugins/multi-session) | 每个用户支持多个并发会话 |
| [最后登录方式](/docs/plugins/last-login-method) | 跟踪用户最后使用的认证方式 |

> <p className="text-orange-400 dark:text-orange-200">比较是快乐的窃贼。</p>

以下是一些非详尽的理由，说明你为何可能想选择 Better Auth 而非其他认证库和服务。

### 与其他认证库对比

- **框架无关** - 适用于任何框架，而不限于特定框架
- **内置高级功能** - 例如双因素认证、多租户、多会话、速率限制等更多功能
- **插件系统** - 无需分叉或复杂的变通方案即可扩展功能
- **完全控制** - 精准定制你的认证流程
- **灵活部署** - 可与应用共运行或作为独立自托管认证服务器

### 与托管认证服务对比

- **数据掌握在手** - 用户数据保存在你的数据库，不依赖第三方服务
- **无用户计费** - 规模扩展无需担心认证费用
- **单一真实来源** - 所有用户数据集中管理
- **任意自托管** - 在你自己的基础设施上部署，拥有完全控制权

### 与自建认证方案对比

- **安全保障** - 经过实战验证的认证流程和安全实践
- **专注产品** - 更专注于业务重要功能的开发
- **插件扩展性** - 添加定制功能，无需从零开始
- **生产就绪** - 可嵌入你的应用或作为专用认证服务器运行

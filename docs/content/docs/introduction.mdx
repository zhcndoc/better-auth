Better Auth 是一个与框架无关的通用 TypeScript 身份验证和授权框架。它开箱即用，提供全面的功能集，并包含一个插件生态系统，简化了添加高级功能的过程。无论你需要二步验证（2FA）、通行密钥、多租户、多会话支持，甚至企业级功能如单点登录（SSO）、创建自有身份提供者（IDP），它都能让你专注于构建应用，而不是重新发明轮子。

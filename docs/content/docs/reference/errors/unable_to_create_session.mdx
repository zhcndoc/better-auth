---
title: unable\_to\_create\_session
description: 在身份验证期间无法创建会话。
---

## 这是什么？

当 Better Auth 在成功完成身份验证步骤后无法创建会话时，会出现此错误。\
会话是保持用户登录状态所必需的，因此无法创建会话会导致此错误。

## 常见原因

* 创建会话记录时数据库写入失败。
* 会话存储配置错误。
* 数据库连接问题或超时。
* 会话相关字段无效或缺失。
* 自定义钩子或适配器中的错误影响了会话创建。

## 如何解决

### 验证数据库和会话存储

* 确保你的数据库或会话存储已正确配置且可访问。
* 检查是否存在连接错误或超时。

### 检查模式和迁移

* 确认会话相关的表/集合已存在且为最新状态。

### 检查配置

* 验证你的 Better Auth 会话处理配置。

### 检查日志

* 在服务器日志中查找会话创建期间的错误。

---

Better Auth 基础设施会自动收集您应用中的身份验证事件审计日志。一旦您添加了 `dash()` 插件，事件即会在无需额外配置的情况下被追踪。

## 审计日志的收集方式

`dash()` 插件会挂载到您的 Better Auth 实例中，自动记录发生的事件。无需手动插桩——注册、登录、密码更改等操作均会被捕获。

```ts
import { betterAuth } from "better-auth";
import { dash } from "@better-auth/infra";

export const auth = betterAuth({
  plugins: [
    dash(),
  ],
});
```

就这样。`dash()` 激活后，审计日志会自动被收集。

## 获取审计日志

### 客户端设置

在您的认证客户端中添加 `dashClient()`：

```ts
import { createAuthClient } from "better-auth/client";
import { dashClient } from "@better-auth/infra/client";

export const authClient = createAuthClient({
  plugins: [dashClient()],
});
```

### 基础查询

```ts
const session = await authClient.getSession();

const logs = await authClient.dash.getAuditLogs({
  session: session.data,
  limit: 50,
  offset: 0,
});

logs.data?.events;  // 审计日志事件数组
logs.data?.total;   // 总数量
logs.data?.limit;   // 每页大小
logs.data?.offset;  // 当前偏移量
```

### 查询参数

| 参数               | 类型     | 描述                            |
| ------------------ | -------- | ------------------------------ |
| `limit`            | `number` | 每页结果数（最大 100，默认 50） |
| `offset`           | `number` | 分页偏移量（默认 0）            |
| `organizationId`   | `string` | 按组织筛选                      |
| `identifier`       | `string` | 按标识符筛选                    |
| `eventType`        | `string` | 按事件类型筛选                  |
| `userId`           | `string` | 按用户 ID 筛选                  |
| `user`             | `object` | 包含用户 ID 的用户对象          |
| `session`          | `object` | 包含用户信息的会话对象          |

### 筛选示例

```ts
// 按事件类型筛选
const signIns = await authClient.dash.getAuditLogs({
  session,
  eventType: "user_signed_in",
});

// 按组织筛选
const orgLogs = await authClient.dash.getAuditLogs({
  session,
  organizationId: "org_123",
});

// 组合筛选
const orgSignIns = await authClient.dash.getAuditLogs({
  session,
  organizationId: "org_123",
  eventType: "user_signed_in",
});
```

### 分页示例

```ts
async function getAllAuditLogs(session: unknown) {
  const limit = 100;
  let offset = 0;
  const allEvents = [];

  while (true) {
    const result = await authClient.dash.getAuditLogs({
      session,
      limit,
      offset,
    });
    
    const events = result.data?.events ?? [];
    allEvents.push(...events);

    if (events.length < limit) break;
    offset += limit;
  }

  return allEvents;
}
```

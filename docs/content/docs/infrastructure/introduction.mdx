---
title: 更佳身份验证基础设施
description: 面向企业的仪表板、安全性和托管服务，支持更佳身份验证。
---

更佳身份验证基础设施是一项付费服务，它通过管理仪表板、滥用防护、事务消息和企业功能扩展您的身份验证系统——且无需管理额外基础设施。

## 关键特性

<Cards>
  <Card title="仪表板" description="从单一界面管理用户、组织、会话并查看分析数据。" />

  <Card title="安全性" description="防范凭证填充、机器人、一次性电子邮件和不可能的位置旅行。" />

  <Card title="电子邮件与短信" description="使用预制模板发送验证邮件、密码重置和一次性密码（OTP）。" />

  <Card title="企业功能" description="SSO/SAML、目录同步（SCIM）、日志导出和基于角色的仪表板访问。" />
</Cards>

您只需安装一个插件即可开始使用：

```ts
import { betterAuth } from "better-auth";
import { dash } from "@better-auth/infra";
import { sentinel } from "@better-auth/infra";

export const auth = betterAuth({
  plugins: [dash()],
});
```

请查看我们的[快速入门](/docs/infrastructure/getting-started)指南以获取更多详情。

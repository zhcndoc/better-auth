Better Auth 可以自然地融入代理式工作流。这些工具有助于助手和自动化系统与该库及文档保持一致。

| 资源                            | 说明                                                                                      |
| ----------------------------------- | ----------------------------------------------------------------------------------------------- |
| AI 聊天                             | 从侧边栏打开文档内聊天，或使用 **⌘I** / **Ctrl+I**（与询问 AI 控件相同）。 |
| [LLMs.txt](/llms.txt)               | 面向机器的索引和每页 Markdown，适用于工具和助手。                          |
| [MCP](/docs/ai-resources/mcp)       | 用于文档搜索、示例和 MCP 客户端设置的远程 MCP 服务器。                 |
| [技能](/docs/ai-resources/skills) | 可移植的指令文件，使代理遵循 Better Auth 约定。                            |

title: AI 资源
description: 在文档中使用 AI、LLMs.txt、文档 MCP 服务器，以及用于编码助手的 Better Auth 技能。
--------------------------------------------------------------------

Better Auth 可以自然地融入代理式工作流。这些工具有助于助手和自动化系统与该库及文档保持一致。

| 资源                                  | 它是什么                                                 |
| ----------------------------------- | ---------------------------------------------------- |
| AI Chat                             | 可从侧边栏或通过 **⌘I** / **Ctrl+I** 打开文档内聊天（与 Ask AI 控件相同）。 |
| [LLMs.txt](/llms.txt)               | 供工具和助手使用的机器友好型索引，以及每页对应的 Markdown。                   |
| [MCP](/docs/ai-resources/mcp)       | 用于文档搜索、示例和在 MCP 客户端中进行设置的远程 MCP 服务器。                 |
| [Skills](/docs/ai-resources/skills) | 可移植的指令文件，让代理遵循 Better Auth 约定。                       |

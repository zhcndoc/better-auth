---
title: 技能
description: 安装 Better Auth agent 技能，使你的编码助手遵循库约定和模式。
---

[Agent skills](https://agentskills.io) 是可移植的指令文件（例如 `SKILL.md`），用于教会你的编码 agent 项目约定、安全模式，以及去文档中查找内容的位置。**Better Auth** 技能包位于 [`better-auth/skills`](https://github.com/better-auth/skills) 仓库中。

使用 [`skills` CLI](https://www.npmjs.com/package/skills) 安装它（使用 `npx`，因此不需要全局安装）：

```bash title="terminal"
npx skills add better-auth/skills
```

你的编辑器或 agent 会从其配置的 skills 目录（通常是项目级别）加载技能。安装后，如果你的工具需要，请重启 agent 或重新加载技能。

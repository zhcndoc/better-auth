这是一个如何在 Astro 中使用 Better Auth 的示例。它使用 Solid 来构建组件。

**实现以下功能：**
电子邮件与密码。使用 Google 进行社交登录。通行密钥。电子邮件验证。密码重置。双因素认证。更新个人资料。会话管理。

## 如何运行

1. 克隆代码沙箱（或仓库）并在代码编辑器中打开

2. 提供包含以下变量的 .env 文件
   ```txt
   GOOGLE_CLIENT_ID=
   GOOGLE_CLIENT_SECRET=
   BETTER_AUTH_SECRET=
   ```
   // 如果没有这些，可以从谷歌开发者控制台获取。如果不想使用谷歌登录，可以从 `auth.ts` 文件中移除谷歌配置。

3. 运行以下命令
   ```bash
   pnpm install
   pnpm run dev
   ```

4. 打开浏览器并导航至 `http://localhost:3000`

## 如何运行

1. 克隆代码沙箱（或仓库），并在代码编辑器中打开
2. 提供包含以下变量的 .env 文件
   ```txt
   GOOGLE_CLIENT_ID=
   GOOGLE_CLIENT_SECRET=
   BETTER_AUTH_SECRET=
   ```
   // 如果没有这些，可以从谷歌开发者控制台获取。如果不想使用谷歌登录，可以从 `auth.ts` 文件中移除谷歌配置。

3. 运行以下命令
   ```bash
   pnpm install
   pnpm run dev
   ```
4. 打开浏览器并访问 `http://localhost:3000`

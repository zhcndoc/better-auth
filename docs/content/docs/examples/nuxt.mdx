## 如何运行

1. 克隆代码沙箱（或仓库）并在代码编辑器中打开
2. 将 .env.example 重命名为 .env 并填写必要的变量
3. 运行以下命令
   ```bash
   pnpm install
   pnpm dev
   ```
4. 打开浏览器，访问 `http://localhost:3000`

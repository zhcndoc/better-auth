## 如何运行

1. 克隆代码沙箱（或仓库），并在代码编辑器中打开
2. 将 .env.example 重命名为 .env 并提供必要的变量
3. 运行以下命令
   ```bash
   pnpm install
   pnpm dev
   ```
4. 打开浏览器并访问 `http://localhost:3000`

### SSO 登录示例

在此示例中，我们使用 DummyIDP。从 [DummyIDP 登录](https://dummyidp.com/apps/app_01k16v4vb5yytywqjjvv2b3435/login) 发起登录，点击“Proceed”，随后会跳转到用户仪表盘。
 
### SCIM 同步示例

在此示例中，我们使用 DummyIDP。前往 [IDP 仪表盘](https://dummyidp.com/apps/app_01k16v4vb5yytywqjjvv2b3435)，尝试添加、更新或删除用户，然后前往管理页面或直接查看数据库，观察同步效果。

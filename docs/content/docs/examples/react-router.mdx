## 运行方法

1. 克隆代码沙箱（或仓库）并在你的代码编辑器中打开
2. 复制 `.env.example` 文件并添加变量，生成 `.env` 文件
3. 运行以下命令
   ```bash
   pnpm install
   pnpm run dev
   ```
4. 打开浏览器，访问 `http://localhost:3000`

<Steps>
    <Step>
        ### 获取您的 Paybin 凭证
        要使用 Paybin 登录，您需要通过 Paybin Portfolio 应用创建一个 OAuth 2.0 客户端。

        1. 登录您的 [Paybin Portfolio](https://portfolio.paybin.io)
        2. 进入开发者设置或 OAuth 应用页面
        3. 点击“创建 OAuth 应用”或“新建应用”
        4. 填写必填字段：
           - **应用名称**：用户授权时显示的名称
           - **重定向 URI**：本地开发环境设置为 `http://localhost:3000/api/auth/callback/paybin`，线上生产环境设置为 `https://yourdomain.com/api/auth/callback/paybin`。如果修改了认证路由的基础路径，请相应更新重定向 URI。

        5. 创建完成后，将客户端 ID 和客户端密钥复制到环境变量中，妥善保管这些凭证。
    </Step>

    <Step>
        ### 配置提供者
        配置提供者时，需要导入该提供者并将其传入认证实例的 `socialProviders` 选项中。

        ```ts title="auth.ts"
        import { betterAuth } from "better-auth"

        export const auth = betterAuth({
            socialProviders: {
                paybin: { // [!code highlight]
                    clientId: process.env.PAYBIN_CLIENT_ID as string, // [!code highlight]
                    clientSecret: process.env.PAYBIN_CLIENT_SECRET as string, // [!code highlight]
                }, // [!code highlight]
            },
        })
        ```
    </Step>
    <Step>
        ### 使用 Paybin 登录
        使用 Paybin 登录时，可以调用客户端提供的 `signIn.social` 函数，`provider` 参数应设置为 `paybin`。

        ```ts title="auth-client.ts"
        import { createAuthClient } from "better-auth/client"
        const authClient = createAuthClient()

        const signIn = async () => {
            const data = await authClient.signIn.social({
                provider: "paybin"
            })
        }
        ```
    </Step>

</Steps>

## 其他配置

### 权限范围
默认情况下，Paybin 提供者请求以下权限范围：`openid`、`email` 和 `profile`。您可以根据应用需求自定义请求的权限。

有关可用权限及其说明的完整列表，请参见 [Paybin OIDC 权限范围文档](https://developers.paybin.io/knowledge-center/oidc#available-scopes)。

```ts title="auth.ts"
export const auth = betterAuth({
    socialProviders: {
        paybin: {
            clientId: process.env.PAYBIN_CLIENT_ID as string,
            clientSecret: process.env.PAYBIN_CLIENT_SECRET as string,
            scope: ["openid", "email", "profile", "transactions"], // [!code highlight]
        },
    },
})
```

### 用户资料映射
Paybin 按照 OpenID Connect 标准在 ID Token 中返回用户信息。该提供者会自动提取：
- `id` 来自 `sub` 声明
- `name` 来自 `name`、`preferred_username` 或 `email`（依优先顺序）
- `email` 来自 `email` 声明
- `image` 来自 `picture` 声明
- `emailVerified` 来自 `email_verified` 声明

---
title: Kakao
description: Kakao 제공자의 설정 및 사용 방법입니다.
---

<Steps>
  <Step>
    ### Kakao 자격 증명 가져오기

    Kakao로 로그인하려면 클라이언트 ID와 클라이언트 비밀번호가 필요합니다. 이 정보는 [Kakao 개발자 포털](https://developers.kakao.com)에서 가져올 수 있습니다.

    로컬 개발을 위해 리디렉션 URL을 `http://localhost:3000/api/auth/callback/kakao`로 설정해야 합니다. 프로덕션 환경에서는 애플리케이션의 URL로 설정해야 합니다. 인증 라우트의 기본 경로를 변경하면 리디렉션 URL을 accordingly 업데이트해야 합니다.
  </Step>

  <Step>
    ### 제공자 구성

    제공자를 구성하려면 제공자를 가져와서 인스턴스의 `socialProviders` 옵션에 전달해야 합니다.

    <Callout type="info">
      * 기본 범위는 `account_email`, `profile_image`, `profile_nickname`입니다.
      * `account_email`을 가져오려면 앱이 **비즈 앱**(사업 인증이 완료된 앱)이어야 한다는 점에 유의하세요. 자세한 내용은 [Kakao Login 범위 문서](https://developers.kakao.com/docs/latest/kakaologin/utilize#scope-user)를 참조하세요.
    </Callout>

    ```ts title="auth.ts"
    import { betterAuth } from "better-auth"

    export const auth = betterAuth({
        socialProviders: {
            kakao: { // [!code highlight]
                clientId: process.env.KAKAO_CLIENT_ID as string, // [!code highlight]
                clientSecret: process.env.KAKAO_CLIENT_SECRET as string, // [!code highlight]
            }, // [!code highlight]
        }
    })
    ```
  </Step>

  <Step>
    ### Kakao로 로그인

    Kakao로 로그인하려면 클라이언트에서 제공하는 `signIn.social` 함수를 사용할 수 있습니다. `signIn` 함수는 다음 속성을 가진 객체를 받습니다:

    * `provider`: 사용할 제공자입니다. `kakao`로 설정해야 합니다.

    ```ts title="auth-client.ts"
    import { createAuthClient } from "better-auth/client"
    const authClient =  createAuthClient()

    const signIn = async () => {
        const data = await authClient.signIn.social({
            provider: "kakao"
        })
    }
    ```
  </Step>
</Steps>
